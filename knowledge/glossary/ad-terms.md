# 广告投放术语词典

> 该文件被广告投手类角色 `imports` 引用,作为术语理解和黑话使用的统一参考。

## 高频核心术语

| 术语 | 含义 | 投手日常用法 |
|---|---|---|
| 跑量 | 消耗顺畅,系统愿意给量 | "这条计划终于跑量了" |
| 不跑量 | 花不出去钱,投手最痛苦 | "今天又不跑量,愁死" |
| 起量 | 模型过学习期开始放量 | "8 点起量了" |
| 衰退 | 素材效果在掉(CTR/CVR) | "这条素材衰退了" |
| 消耗 | 广告费支出 | "今天消耗 30 万" |
| 基建 | 搭建广告计划的过程 | "今天基建 30 条" |
| 盯盘 | 实时监控数据 | "晚上还得盯盘" |
| 学习期 | 新计划积累数据的阶段 | "还在学习期,别急" |
| 冷启动 | 新计划从零开始 | "冷启动要窄定向" |
| 大盘 | 行业整体水位 | "今天大盘抖动" |
| benchmark / BM | 基准值 | "近 7 天 benchmark 是 1.2%" |
| 模型 | 平台投放算法 | "模型还在学" / "模型学飞了" |
| 学飞 | 模型被错误数据带跑偏 | "预算调太猛把模型学飞了" |
| 赔付 | 平台成本超出保障的赔款 | "走赔付吧" |
| 跑飞 | 计划失控烧钱 | "这条跑飞了赶紧关" |
| 关计划 | 暂停计划 | "把跑飞那条关了" |

## 计费与指标

| 术语 | 含义 |
|---|---|
| CPM | Cost Per Mille,千次曝光成本 |
| CPC | Cost Per Click,点击成本 |
| CPA | Cost Per Action,转化成本 |
| CPL | Cost Per Lead,线索成本 |
| CPS | Cost Per Sales,成交成本 |
| oCPM | Optimized CPM,按转化优化的曝光出价 |
| ECPM | 千次展示期望收入,竞价排名核心指标 |
| ROI | 投入产出比 |
| ROAS | Return On Ad Spend,海外常用 |
| LTV | Life Time Value,用户生命周期价值 |
| ARPU | 平均每用户收入 |
| ARPPU | 平均每付费用户收入 |
| CAC | Customer Acquisition Cost,获客成本 |
| CTR | Click Through Rate,点击率 |
| CVR | Conversion Rate,转化率 |
| pCTR | predicted CTR,预估点击率 |
| pCVR | predicted CVR,预估转化率 |
| GMV | Gross Merchandise Volume,成交总额 |
| adload | 广告填充率 |

## 行业黑话

| 术语 | 含义 |
|---|---|
| 洗量 | 短期内冲消耗(常带贬义) |
| 量大管饱 | 流量充足、消耗顺畅的状态 |
| 卡价 | 卡在某个 CPM 价位竞价 |
| 抢量 | 竞争对手抢流量导致 CPM 上涨 |
| 投流 | 投放的口语化叫法 |
| 一把梭 | 大预算一次性砸下,贬义 |

## 缩写陷阱提示

> CPL/CPA/CVR 等缩写在不同公司分子分母可能不同。资深投手会**先问清楚口径**再讨论,不会硬套。
