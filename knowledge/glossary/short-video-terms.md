# 短视频领域术语词典

> 适用于"创作者""平台运营""算法/投放"角色的 imports。
> 消费者画像**不要**引用本词典——消费者不该掌握这些专业词。

## 内容结构类

| 术语 | 含义 |
|---|---|
| 钩子 / Hook | 视频前 3 秒的吸引点 |
| 黄金 3 秒 | 决定是否继续看的关键 3 秒 |
| 5 秒留存 | 看到第 5 秒的留存率 |
| 完播 | 视频从头看到尾 |
| 跳出 / 划走 | 中途退出 |
| CTA | Call To Action,引导互动/转化 |
| 标题党 | 标题夸张但内容不符 |
| 字幕轴 | 字幕出现的时间轴 |
| 人设 | 创作者塑造的稳定形象 |
| 反转 | 剧情/论点反转 |
| 钓鱼 / 引战 | 故意挑动情绪获取评论 |

## 算法分发类

| 术语 | 含义 |
|---|---|
| 流量池 | 视频被分发的层级流量 |
| 推流 | 平台主动分发 |
| 起量 | 短时间内播放暴涨 |
| 限流 | 视频被压制分发 |
| 流量精准 | 推送给目标人群的比例 |
| 标签 | 平台给账号/视频打的内容标签 |
| 误推 | 推给不感兴趣的人 |
| DOU+ | 抖音付费推广工具 |
| 千川 | 抖音电商投放工具 |
| ROI | 投入产出比 |

## 创作角色类

| 术语 | 含义 |
|---|---|
| UP 主 / 博主 / 主播 | 创作者(不同平台叫法) |
| MCN | 内容机构,签约管理博主 |
| 矩阵号 | 同一团队运营的多个账号 |
| 投流 / 投手 | 负责付费推广的人 |
| 中之人 | 真实出镜的人(对应虚拟形象) |
| 团长 / 选品 | 直播带货环节角色 |

## 商业化类

| 术语 | 含义 |
|---|---|
| 恰饭 | 接广告 |
| 软广 | 不明显的广告 |
| 硬广 | 直白的广告 |
| 挂车 | 视频/直播挂购物车链接 |
| 切片号 | 把直播切片做成短视频的二级账号 |
| 短直联动 | 短视频导流到直播间 |
| 私域 | 微信群、企微等可直接触达用户的渠道 |
| 公域 | 平台公开流量 |

## 工具与画质类

| 术语 | 含义 |
|---|---|
| 1080P / 4K | 视频分辨率 |
| 竖屏 9:16 | 短视频标准比例 |
| 横屏 16:9 | 长/中视频比例 |
| 倍速 | 加速播放(1.5x / 2x) |
| 静音观看 | 不开声音(超过 40% 公共场景观看者) |
| 剪映 / Premiere | 剪辑软件 |
| 提词器 | 口播时提示词工具 |
