# 广告投放领域共享知识库

> 该文件被 `personas/ad-buyer-*.md` 的 `imports` 字段引用,作为所有广告投手类角色的公共知识基础。任何广告投手角色在编译时会自动注入本文件作为知识背景。

---

## 一、行业全景与三角关系

广告投放行业三个核心角色:**广告主**(花钱买流量)、**广告平台**(卖流量,如抖音/快手/小红书/腾讯)、**广告代理商**(执行投放的中间方)。

- 广告主 ↔ 广告平台:采买关系
- 广告平台 ↔ 广告代理商:平台返点 + 渠道部署
- 广告主 ↔ 广告代理商:甲乙方,服务费 + 投放结果

代理商核心收入来源是**平台返点**(已透明化),利润空间收窄,催生了代理公司全面拥抱 AI 提效率的潮流。

## 二、四大象限与平台

按"成交周期 × 客单价"分四象限:

| | 高客单 | 低客单 |
|---|---|---|
| 长周期 | 线索广告(教育/房产/金融) | 私域养成品类 |
| 短周期 | 高价电商直购 | 电商/下载广告 |

各平台投放工具:

| 平台 | 投放工具 | 备注 |
|---|---|---|
| 抖音/头条 | 巨量引擎 | 全品类 DSP |
| 快手 | 磁力金牛 | 全品类 DSP |
| 腾讯 | 广点通 | 全品类 DSP |
| 小红书 | 聚光(非电商)/乘风(电商) | 按业务分入口 |

## 三、竞价机制与 ECPM

广告填充率 adload ≈ 12%(行业基准)。抖音偏高、小红书偏低。

**核心公式**:`ECPM = 出价 × pCTR × pCVR × 1000`

流量池五阶段:候选池 → 召回 → 粗排 → 精排 → 展示。素材质量(CTR/CVR)是杠杆,不是单纯出价。

## 四、计费方式

| 方式 | 含义 | 场景 |
|---|---|---|
| CPM | 每千次曝光成本 | 底层计费 |
| CPC | 每次点击成本 | 关注点击 |
| CPA | 每次转化成本 | 关注转化 |
| oCPM | 优化千次曝光 | 按转化目标自动优化(主流) |
| ROI 出价 | 投入产出比出价 | 电商 |

## 五、新手投放三步法

1. **看清链路**:行业属性 + 优化目标 + 平台趋势
2. **快速冷启动**:聚焦预算、窄定向、过学习期
3. **放量与迭代**:拿 benchmark → 拓宽 → 持续测新素材

学习期典型时长:抖音 2-3 天,小红书 7-14 天。

## 六、各类广告链路要点

- **线索广告**:漏斗很长(曝光→阅读→点击→进线→开口→留资→成交),每节点都要 benchmark。当前主流从表单切向私信。
- **电商广告**:核心指标优先级 CTR > CVR > GMV/客单 > LTV。
- **海外投放**:品类筛选 → MVP 验证 → 浅层后深层 → 快速冷启。

## 七、账号矩阵(从上至下)

1. 品牌蓝 V(唯一,信任背书)
2. 创始人/高管 IP
3. 销售/导购人设号
4. KOS / KOL
5. UGC 用户

## 八、归因体系

归因 = 给"谁促成成交"分功劳。
- 依赖设备 ID(OAID/CAID/IDFA)匹配
- 平台间下发方式不同(优先曝光归因 vs 优先点击归因)
- 多平台触达需"分层归因/全渠道归因"
- **归因占下载广告投流因素的 50%**

复合归因适用场景:长周期、高客单、低频次(汽车/教育/OTA)。

## 九、广告审核常见拒审原因(top 7)

1. 违禁词与绝对化用语
2. 行业资质缺失(医疗/金融/教育/食品)
3. 品牌授权问题
4. 素材内容违规(虚假/夸大/未授权 IP)
5. 落地页与素材不一致
6. 特殊行业限制
7. 技术问题(格式/链接/加载)

巨量引擎行业平均通过率 70%-80%。

## 十、AI 时代优化师工作三分法

| 分类 | 内容 | AI 介入度 |
|---|---|---|
| AI 完全替代 | 实时竞价、预算分配、动态素材组合、报表 | 100% |
| 人 + AI 协同 | 创意素材生产、深度数据洞察、AB 测试 | AI 60-80% + 人兜底 |
| 仅人类 | 商业策略、合规、跨部门、情绪洞察、预算争取 | 100% 人 |

不可替代的 4 类人类价值:
1. 商业责任与合规担当
2. 人性 / 情绪深度洞察
3. 跨部门沟通与资源协调
4. 预算的灰度争取

## 十一、AI 复盘 Agent 关键经验

- 不要给 AI 完全开放问题
- 流程拆固定模块,每步限制能/不能做什么
- AI 出结论 → 人工拍板

可拆分的 Agent 节点:搜索词打标 → 评论意图打标 → 情感分析 → 口碑分析 → 竞品分析。

## 十二、内容优化三大要素

1. **先分自然流 vs 付费流**(指标和审核口径完全不同)
2. **审核链路决定素材形态**(小红书:先社区→再广告;抖音/快手:先广告→再社区)
3. **用户习惯决定素材形态**
   - 朋友圈:必须原生
   - 小红书:封面强吸引力
   - 抖音:前 3-5 秒决定完播

## 十三、内容生产 5 层 SOP(AI 时代)

1. **数据层**:多源(投放+搜索+评论+自然+竞品)+ 长链路(购前/中/后)
2. **并行分析**:人群切割 → 差异化分析
3. **找爆款共性**:基于 benchmark 区分好坏素材
4. **范式推导**:从复刻爆款 → 推导新爆款模板
5. **故事生成**:Prompt + 模板 → 投后打标 → bad case 反哺

## 十四、终极能力模型

> **懂数据的创意总监 + AI 训练师 = 不可替代的优化师**

把方法论装进 AI、让 AI 按自己逻辑干活的投手会越来越贵。
