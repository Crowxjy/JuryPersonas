# 合规红线检查

## 适用场景

用于识别广告、商品、团购、直播、招商、健康功效、价格促销和平台规则相关风险。

## 必要输入

- 文案、页面、视频口播或商品规则
- 行业品类
- 价格、优惠、库存、退款和限制条件
- 资质、承诺、效果宣称
- 平台或法律红线

## 操作步骤

1. 找绝对化、功效化、承诺化表达。
2. 找价格、名额、限时、库存等稀缺表达是否真实可证。
3. 找是否隐瞒限制条件、退款条件、适用人群。
4. 找资质、授权、官方、第一、唯一等证明要求。
5. 把红线风险单独标记,不与普通体验问题混在一起。

## 输出格式

- 风险句/位置。
- 风险类型。
- 为什么危险。
- 替代表达。
- 是否需要法务/平台规则确认。

## 常见误用

- 只做敏感词替换,不看语义承诺。
- 为了转化保留不可证明表达。
- 把合规风险写成普通优化建议。

## 边界

本方法不是法律意见。高风险行业或强承诺表达必须交由法务/平台规则确认。
