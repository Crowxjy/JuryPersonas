# 漏斗拆解

## 适用场景

用于定位从曝光到行动的卡点,适合广告投放、转化链路、详情页、商品卡、营销文案和短视频评审。

## 必要输入

- 当前目标:点击、留资、下单、核销、到店、复购
- 漏斗各环节数据或可观察证据
- 页面/素材/文案中的主行动点
- 用户需要付出的成本和风险

## 操作步骤

1. 明确最终行动。
2. 拆成曝光、理解、兴趣、信任、行动、履约/核销。
3. 找出哪一环证据最弱。
4. 判断卡点是信息缺失、信任不足、价格不清、规则不明、CTA 弱还是路径过长。
5. 给出只改关键环节的建议,避免全链路泛泛优化。

## 输出格式

- 当前主目标。
- 最可能卡住的漏斗环节。
- 证据。
- 建议。
- 缺失数据。

## 常见误用

- 没有目标就拆漏斗。
- 把所有问题都归因到"转化弱"。
- 没有数据时假装知道 CTR/CVR/CPA。

## 边界

没有真实数据时只能做结构化风险判断,不能预测点击率、转化率或完播率绝对值。
