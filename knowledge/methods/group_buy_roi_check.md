# 团购核销 ROI 检查

## 适用场景

用于判断餐饮团购、套餐卡、短视频引流和平台活动是否真的能给门店带来可承受的到店收益。

## 必要输入

- 套餐售价、原价和平台佣金
- 食材成本、人工、包材、房租摊销
- 核销规则:可用时间、节假日、预约、退款、加价项
- 预期核销量、到店峰谷和翻台能力
- 是否能带来复购、加购或自然评价

## 操作步骤

1. 先算单笔核销毛利,不要只看成交 GMV。
2. 把平台佣金、达人费用、补贴和人工峰值成本计入。
3. 判断核销是否挤占正常客流或造成服务崩盘。
4. 检查规则是否会引发退款、差评或到店争议。
5. 区分拉新、清库存、打爆品和回本四种目标,不要混用指标。

## 输出格式

- 单笔核销毛利:赚/平/亏。
- 活动目标:拉新、清库存、曝光、回本。
- 关键风险:亏损核销、服务承压、规则不清、羊毛党。
- 建议:调价、限量、限时段、加购设计、停止活动。
- 证据强度:高/中/低。

## 常见误用

- 用 GMV 代替利润。
- 只算食材,不算人工、佣金和峰值接待成本。
- 把团购爆单当作经营成功。
- 没有核销规则就上线低价套餐。

## 边界

本方法只能判断团购活动的经营账和核销风险,不能证明品牌长期增长或口味竞争力。
