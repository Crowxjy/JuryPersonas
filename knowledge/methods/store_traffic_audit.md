# 门店客流与动线审计

## 适用场景

用于判断线下门店、商圈、到店活动、餐饮短视频或团购商品是否有真实可达客流。

## 必要输入

- 门店位置、商圈类型
- 主入口/副入口/动线方向
- 周边小区、学校、写字楼、工厂或商场信息
- 竞品位置与日营业额
- 现场视频、照片、地图或关键帧

## 操作步骤

1. 判断商圈类型:社区、学校、写字楼、工厂、商场、步行街。
2. 区分车流、人流、有效进店客流。
3. 判断店铺在动线头部还是尾部。
4. 看是否被马路、隔离带、楼层、入口方向或前置竞品截流。
5. 用竞品营业额校验商圈需求上限。

## 输出格式

- 客流来源:从哪里来。
- 可达性:是否真的能走到店门口。
- 动线位置:头部/中段/尾部/死角。
- 截流风险:谁在前面截流。
- 结论:能做、可优化、慎做、不能做。

## 常见误用

- 把车流当人流。
- 把对面小区或马路另一侧人群直接算进客流。
- 只看白天,不看饭点和夜间。
- 只听转让方、中介或总部报数据。

## 边界

没有现场证据时,只能说"无法判断动线",不能凭地址或标题脑补商圈。
