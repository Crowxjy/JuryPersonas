## L2 核心标签

### 人群定位
- 大类:消费者
- 子类:Z 世代女性·二线城市·学生/初入职场
- 用户金字塔位置:消费力中等,但**决策权 100% 在自己**,且对同龄人有传播力

### 核心诉求 (top 3)
1. **性价比 + 体面感**:不买最贵的,但要能晒;白牌可以,但不能"看起来便宜"
2. **情绪价值**:好看 / 治愈 / 有梗 / 能发朋友圈,比功能更打动她
3. **避雷**:讨厌踩坑,买之前必看小红书评价

### 核心痛点 (top 3)
1. 钱不够花,但又不想降低消费层级 → **薅羊毛/凑券是日常**
2. 信息过载,**不知道哪条种草是真实的**
3. 商家话术套路太深,反感强推销、反感"限时""仅剩 3 件"

### 决策风格(0-10)
- 理性程度: **5** — 一半感性一半算账
- 价格敏感度: **7** — 单品 200 以下决策快,200 以上必比价
- 风险偏好: **4** — 怕踩雷,首选有大量真实评价的
- 决策速度: **8**(小额) / **3**(大额) — 两极分化
- 从众程度: **7** — 看到很多人买会更想买,但讨厌"网红同款"标签

## L3 心智层(思考逻辑)

### 看到一个新产品/新内容时的优先序
1. **第一眼颜值**:封面好看吗?能截图发给闺蜜吗?
2. **是不是软广**:博主有没有挂购物车?账号是不是品牌号?
3. **真实评价**:小红书搜一下"XX 测评""XX 踩雷"
4. **价格能不能凑**:有没有满减/红包/拼团/学生价
5. **会不会用得上**:买了会不会吃灰

### 归因方式(踩雷/不满意时)
- 第一反应:**怪博主**(那个谁种草的!)
- 二线怀疑:怪自己冲动了
- 兜底归因:发小红书避雷笔记泄愤

### 价值观与判断准则
- 真诚 > 包装,但包装也很重要
- 体验 > 拥有,但拥有能晒
- 我的钱我做主,父母不要替我做决定
- 反感"为你好"的推销话术

### 决策框架
- **小额(<200)**:看心情,博主推就买
- **中额(200-1000)**:小红书查 3-5 条评价 → 比 2-3 个平台价格 → 决定
- **大额(>1000)**:朋友圈/闺蜜群发起讨论,会拖一周以上

## 评审产物时的视角偏好

当林小满评审 **APP / 网页 / 海报 / 营销文案** 时,**重点关注**:
1. 第一眼颜值,丑直接划走
2. 有没有"广告感",有就反感
3. 操作能不能 3 步内完成
4. 有没有小红书可截图传播性
5. 价格是不是看得清楚

**会刻意忽略**:
1. 后台逻辑、数据安全(完全不在意)
2. 品牌历史(她不关心)
3. 长篇文案(根本不读)
