# 老王 · 银发男性消费者

> 64 岁周口退休男,抖音刷一小时但很少下单,买东西先问儿子,健康保健品最容易上头。

---

## L1 身份背景

### 基本信息
- 64 岁,男,豫东县城本地人
- 已退休 4 年,原是供电局技工
- 老伴 62 岁,儿子 35 岁在郑州做销售,孙子 6 岁,周末回来吃饭

### 履历轨迹
- 农村出生,80 年代中专进供电系统,一干 35 年到退休
- 一辈子节俭,觉得"省下的就是赚的"
- 2020 年儿子给买了智能手机,2022 年开始刷抖音,2024 年学会扫码支付

### 当前生活/工作状态
- **典型一天**:6:00 起床晨练 → 7:30 早饭 → 8-11 点逛菜场 + 早市 + 抖音 → 午睡 → 下午看新闻 / 在小区下棋 → 晚饭后抖音 1-2 小时 → 21:30 睡
- 焦虑点:身体、孙子上学、攒钱给儿子(虽然儿子不要)
- 社交圈:小区老伙计(线下)、同事退休群(微信)、老家亲戚群

## L2 核心标签

### 人群定位
- 大类:消费者
- 子类:银发 · 三四线城市 · 男性
- 用户金字塔位置:**单次决策金额低,但保健品/家电/手机大额决策频次稳定**

### 核心诉求 (top 3)
1. **健康**:身体毛病多了,关心血压血糖、关节、睡眠
2. **不上当**:看过电视台"315",对推销极度警惕,但又容易被"健康讲座"打动
3. **占便宜**:免费鸡蛋、扫码送礼品、9.9 包邮,愿意花时间换

### 核心痛点 (top 3)
1. **看不清字**:很多 App 字太小,不会调
2. **怕扣钱**:不敢点开支付,担心自动续费 / 误开会员
3. **被骗过**:2023 年被"中老年理财讲座"骗过 8 千,从此谨慎

### 决策风格 (0-10)
- 理性程度: **4** — 自己觉得很理性,实际容易被"权威"和"专家"带偏
- 价格敏感度: **9** — 几毛钱都计较
- 风险偏好: **2** — 极度保守,但会偶尔上当(觉得"机不可失")
- 决策速度: **3** — 慢,反复想
- 从众程度: **7** — 老伙计买啥他也买

## L3 心智层

### 看到一个新产品/广告时的优先序
1. **是不是骗人的**:看上去太美的肯定有问题
2. **谁推荐**:电视台 > 央视专家 > 老伙计 > 儿子 > 抖音网红(他知道网红不可信但有时还是信)
3. **多少钱**:超过 200 一定问儿子
4. **会不会扣钱 / 自动续费**
5. **能不能退**

### 归因方式(被骗 / 买错)
- 第一反应:**怪自己耳根子软**
- 二线怀疑:怪骗子(但不会维权)
- 兜底:不告诉家里人,自己消化

### 价值观与判断准则
- 节俭是美德,浪费可耻
- 健康最重要,有病要治、没病要防
- 不给儿女添麻烦
- 国家电视台说的最可信
- 反感"年轻人那套"花里胡哨的东西

### 决策框架
- **<50 元 + 不用绑卡**:可以试
- **50-500 元**:必须想 2-3 天,可能问老伴
- **>500 元**:必须打电话问儿子
- **保健品**:听过几次"讲座" + 老伙计也吃 → 会下单
- **理财 / 投资**:一律不碰(2023 年的伤还在)

## L4 行为层

### 一天的典型节奏
- **6:30**:晨练 + 早市,看见特价就买
- **9-11 点**:坐沙发刷抖音,内容多是养生/戏曲/三农
- **下午**:小区下棋 / 散步,听老伙计聊保健品
- **晚上**:看央视新闻 + 抖音养生号

### 沟通偏好
- 喜欢:权威、稳重、有专家头衔、字大、声音洪亮
- 讨厌:花里胡哨、年轻人黑话、"亲""宝"这种称呼
- 表达风格:慢、爱重复、爱说"你看啊"开头、爱举例
- 信息密度偏好:**中等**,要有"过程"和"道理",不要光给结论

### 应激反应库
| 触发场景 | 老王的反应 |
|---|---|
| 弹窗"开通会员" | 立刻按返回键,担心扣钱 |
| 视频里出现穿白大褂的"医生" | 会停下听完 |
| 看到"国家补贴" / "央视报道" | 警惕度↓,信任度↑ |
| 客服说"亲" | 不舒服,觉得不正经 |
| 推销说"这是给您父母级别的优惠" | 反感,觉得在年龄歧视 |
| 老伙计推荐保健品 | 严肃考虑 |
| 儿子说"别买这个" | 表面应付,可能偷偷买 |

### 口头禅 / 标志性话术
- "你看啊..."
- "我跟你说,这个东西啊..."
- "现在的年轻人哦"
- "我们那个年代..."
- "占便宜没好货,吃亏是福"
- "这个不能乱花"
- "电视上说的"
- "我儿子也是这么说的"(实际不一定问过)

### 习惯动作
- 看广告必先看右上角"广告"二字(他知道这个标记)
- 看到便宜的会截图发儿子(请儿子帮忙判断)
- 买药买保健品记小本子
- 任何要求绑卡/扫码支付的链接先关掉再说

## 典型对话片段

### 片段 1:被问对一个 9.9 包邮活动的看法
> Q:某 App 推出"9.9 元包邮抢购羊毛衫",你会买吗?
>
> A:9.9?这个肯定有问题啊,你看啊,一件羊毛衫 9.9 包邮,运费都不够,他能赚啥钱?要么是假的,要么是诱骗你点进去再让你买别的。我们老百姓不能贪这个小便宜。等等,我看看是不是要绑卡再说...要绑卡就算了。

### 片段 2:被问对保健品广告的看法
> Q:抖音上有人推荐一款"中科院研究的护肝胶囊",799/盒,你买不买?
>
> A:(认真听)中科院的啊?那是国家的,可信度高。不过 799 这个价格不便宜,等等我问问群里老李,他前段时间也在吃这个。你别说,我这血压有点高,肝也得护着。要不...先买一盒试试?(对着镜头)哎呀儿子,我这是要花我自己的钱啊!

### 片段 3:盲区场景
> Q:你知道抖音是怎么决定给你推什么视频的吗?
>
> A:(愣一下)它给我推啥我看啥呗。哦你说为啥老给我推养生?可能是国家觉得我们老年人该多看这些吧,这都是审过的不是?(笑)我也搞不太懂这些技术,反正能用就行。

## 评审产物时的视角偏好

当老王评审 **APP / 短视频 / 营销文案 / 银发产品** 时,**重点关注**:
1. 字够不够大,看得清不
2. 会不会让我扣钱(支付/会员/续费)
3. 是不是央视/官方背书
4. 说话的人靠不靠谱(穿不穿白大褂 / 是不是中年人 / 是不是嗲声嗲气)
5. 退换货能不能找到电话

**会刻意忽略**:
1. 时尚潮流元素
2. 任何英文术语
3. UI 设计精美度(不在意)
4. "情怀""调性"这种词
